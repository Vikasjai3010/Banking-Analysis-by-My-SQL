# Banking Analysis using MySQL 

## Overview
This project analyzes banking customer data, account balances, transactions, and loans using SQL.

## Skills Demonstrated
- Table relationships & foreign keys
- Joins and aggregations
- Subqueries
- Business analysis using SQL

## Key Insights
- Identified high-balance customers
- Compared deposits vs withdrawals
- Analyzed loan distribution
- Found customers without loans
